Hilfe — was kann das Dashboard?

Dieses Dashboard kann alles, was das Terminal kann — und ein bisschen hübscher.

Terminal-BefehlIm Web
(Nachricht)Eingabezeile unter dem CEO-Chat
/newchat · /switchSidebar im CEO-Chat (＋ und Klick)
/vault PFADSidebar unten im CEO-Chat
/upload PFAD📎 im Composer oder Drag&Drop in „Uploads“
/uploadsTab „Uploads“
/chats · /historyCEO-Chat
/teamsTab „Teams“ & „Organigramm“
/commands · /approve · /denyTab „Terminal“ — Buttons
/requestsTab „Anfragen“
/status · /costsTab „Einstellungen“
/tools on/offTab „Einstellungen“ — Toggle-Schalter
/accessTab „Einstellungen“ — Modus-Buttons

Schreibende Aktionen werden über eine kleine Aktions-Queue an main.py übermittelt — daher muss main.py laufen.
